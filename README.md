# Omega OS ⚡
A gamer-first, developer-driven custom Android ROM built for raw performance, minimal junk, and full user control.

## 💡 Key Features
- Pre-rooted boot images with KernelSU
- Omega Tweaks: Thermal, FPS boost, GMS nuker
- Based on LineageOS
- DragonBoost 2.0 module support
- Smooth 360Hz touch optimization

## 📱 Supported Devices
- Redmi Note 8 (ginkgo)
- Redmi Note 9 Series (miatoll)
- Redmi Note 8T (willow)
